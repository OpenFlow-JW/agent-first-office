# Agent-First Office

Reduce review/approval bottlenecks in knowledge work with **agentic document reviews** and **codified organizational skills**.

Wedge MVP: **PRD (Markdown) + UX Review Lens**

## Goal
Build an agent-first workflow where documents are treated like code:
- local authoring
- versioning (git)
- PR-based review/approval
- test-case driven definition of done

## Repo structure
- `docs/` PRDs, decisions (ADR), templates
- `api/` OpenAPI specs
- `tests/` Postman/Newman, Playwright, etc.
- `app/` product code (later)
