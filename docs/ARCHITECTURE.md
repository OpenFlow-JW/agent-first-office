# Architecture (Draft)

## Big idea
Treat knowledge-work documents like code:
- local authoring
- version control (git)
- PR-based review/approval
- test-case driven definition of done

## Closed loop
1) Kit (Entry): PRD template + metadata + basic checklist
2) Reviewer (Money): lens-based pre-review report
3) Skill Miner (Moat): mine repeated patterns from review logs into reusable skills

## Current state
This repo is currently scaffolding + templates.

## Near-term plan
- Define PRD wedge + UX lens reviewer report format
- Add example PRDs with synthetic data
- Add a minimal reviewer runner (later)
