# Good first issues

These are small, high-leverage contributions that help the community quickly.

## Docs & templates
- Improve `docs/PRD.template.md` clarity and examples
- Add an example PRD (synthetic) for a simple feature
- Add a UX review lens checklist (generic)

## Process
- Propose repo naming/structure improvements
- Add a glossary for key terms (lens, signoff, skill, miner)

## Tooling (later)
- Add a minimal markdown validator (frontmatter required fields)
- Add a minimal report format validator
