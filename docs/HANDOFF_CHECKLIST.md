# Handoff Checklist (Ian → TK / Engineering)

This is the minimum package before Engineering starts production work.

## Must have
- [ ] PRD filled (use `docs/PRD.template.md`)
- [ ] Acceptance Criteria (test cases) written
- [ ] UX lens review completed (pre-review report attached or summarized)
- [ ] Scope / Non-scope explicit
- [ ] Risks / open questions listed

## Nice to have
- [ ] OpenAPI draft (if API changes)
- [ ] Postman collection draft (if API)
- [ ] Example inputs/outputs (fixtures)
- [ ] Observability notes (logs/metrics)
